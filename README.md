# this-or-that-docs

Public documentation site for the **This or That** Discord bot:
- Privacy Policy
- Terms of Service
- Support / Contact

This repo is designed to be hosted on **GitHub Pages** so the Discord Developer Portal can link to publicly accessible URLs.

## Live URLs (after enabling Pages)
- `/` → Landing page
- `/privacy` → Privacy Policy
- `/terms` → Terms of Service
- `/support` → Support / Contact

## How to enable GitHub Pages
1. Push these files to a public repo named **this-or-that-docs**
2. Go to **Settings → Pages**
3. Under **Build and deployment**
   - **Source**: Deploy from a branch
   - **Branch**: `main` (or `master`) and folder `/ (root)`
4. Save

GitHub will provide a URL like:
`https://<your-username>.github.io/this-or-that-docs/`

## Updating content
Edit any `.md` file and commit to `main`. GitHub Pages will rebuild automatically.
